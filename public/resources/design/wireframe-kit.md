# Mobile App Wireframe Guide (Text-Based)

## 1. Core Screens for MVPs
- **The Splash:** Branding, tagline, Sign In / Sign Up buttons.
- **The Dashboard:** User stats, primary action button (e.g., "New Chat"), and navigation.
- **The Core Action:** The primary feature screen (e.g., AI Chat, Map, Dashboard).
- **The Profile:** User settings, billing, and logout.

## 2. Navigation Patterns
- **Tab Bar (Bottom):** Fix 3-5 icons at the bottom for easy reach.
- **Fab (Floating Action Button):** Use for the single most important action.
- **Back Button (Top Left):** Always allow users to go back.

## 3. UI Kits (Free in Text)
- Instead of complex Figma kits, use:
  - **Icons:** [Lucide React](https://lucide.dev)
  - **Unsplash:** High-quality photos.
  - **Pexels:** Free stock videos for demo backgrounds.

---
*Tip: Focus on 3 key screens. Trying to build a 20-screen app in 48 hours is a recipe for failure.*
