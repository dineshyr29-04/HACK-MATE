# Hackathon Design Aesthetics Guide

## 1. The "WOW" Rule
- First 10 seconds: A clean, high-contrast hero section determines 50% of the judge's opinion.
- Use glassmorphism (`backdrop-blur`) and subtle gradients for a premium feel.

## 2. Typography
- Use a maximum of 2 font families.
- Establish a clear hierarchy: H1 (Headline) > H2 (Section) > Body > Caption.

## 3. Micro-interactions
- Buttons should have `:hover` and `:active` states.
- Use smooth transitions (`transition-all duration-300`) for all state changes.
- Add subtle entrance animations (e.g., fade-in-up) for content sections.

---
*Tip: When in doubt, more whitespace usually makes things look better, not worse.*
